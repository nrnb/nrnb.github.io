How to Get Started

Here are some popular tutorials to help you get started using NRNB tools and resources:

Introduction to Cytoscape

Visualizing data on networks using custom visual styles and basic manipulation of networks.


Basic Expression Analysis in Cytoscape

Visualizing gene expression data using networks.


Cytoscape Web

Interactive web tool for network visualization.


Cytoscape App Development

Learn how to develop your own custom apps on the Cytoscape platform.

Introduction to WikiPathways

Building, editing and sharing pathway models for data visualization and analysis.


WikiPathways and Cytoscape

Accessing and viewing WikiPathways in Cytoscape.


Open Tutorials

Find a growing catelog of free online tutorials for a variety of open source bioinformatics, including Cytoscape and WikiPathways.


OpenHelix(TM)

A fee-based service providing high quality training materials for Cytoscape.
